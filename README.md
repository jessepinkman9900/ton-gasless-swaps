# ton-gassless-swaps
## Pre-Req
- only [w5](https://github.com/tonkeeper/w5) wallets support gasless transactions
- gasless txn provider [tonconsole](https://docs.tonconsole.com/tonapi/cookbook)

## Tools
- [mise](https://mise.jdx.dev/getting-started.html)

## Usage
```sh
mise install
yarn install
```

```sh
yarn start
```
